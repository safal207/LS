# Orientation Trajectory Integration (Phase 14.2)

## Purpose

Integrate `trajectory_error` from the Trajectory Layer into Orientation Center
as a non-invasive signal.

## Input

- `trajectory_error` (float, 0–1)

## Output

- `trajectory_signal` (float, 0–1)

## Notes

- Not used by Rhythm Engine
- Not used by Coordinator
- No behavior changes
- Prepares Phase 15 (Adaptive Coordinator)
