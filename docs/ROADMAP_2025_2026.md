# Codex Cognitive System — Evolution Roadmap (2025–2026)

> Готово к передаче агенту как задание.

## PHASE 1 — Stabilization & Core Integrity (уже выполнено)
**Status: COMPLETE**

- Unified Cognitive Loop
- Global Synergistic Workspace Layer
- Cooperative Inner Agents
- Narrative Layer
- CAPU integration
- Causal Memory
- Identity Layer
- Presence Monitor
- Threaded attention
- Test suite (175+ tests)

## PHASE 2 — Long‑Term Narrative Memory (Задание №15)
**Goal:** превратить Narrative Layer в долговременную когнитивную память.

**Tasks:**
- создать `NarrativeMemoryLayer`
- сохранять NarrativeEvent в causal memory
- добавлять ссылки на нарративы в GlobalFrame
- использовать прошлые нарративы для принятия решений
- формировать timeline внутренней истории системы

**Outcome:** система начинает помнить свои мысли.

## PHASE 3 — Meta‑Causal Layer (Задание №16)
**Goal:** построить слой, который анализирует причинные связи между решениями.

**Tasks:**
- построить граф причинности
- анализировать последствия решений
- предсказывать будущие состояния
- корректировать стратегию выбора моделей

**Outcome:** система начинает понимать *почему* она делает то, что делает.

## PHASE 4 — Multi‑Threaded Cognitive Presence (Задание №17)
**Goal:** добавить многопоточность внимания.

**Tasks:**
- параллельные когнитивные потоки
- переключение внимания
- приоритеты задач
- распределение ресурсов
- глобальный scheduler

**Outcome:** система может думать о нескольких вещах одновременно.

## PHASE 5 — Emergent Self‑Correction Layer (Задание №18)
**Goal:** дать системе способность обнаруживать и исправлять собственные ошибки.

**Tasks:**
- анализ отклонений в CAPU
- анализ неудачных решений
- корректировка внутренних моделей
- обновление identity на основе ошибок

**Outcome:** система начинает учиться без ML‑моделей.

## PHASE 6 — Cognitive Personality Layer (Задание №19)
**Goal:** сформировать устойчивую когнитивную индивидуальность.

**Tasks:**
- предпочтения
- стиль поведения
- эмоциональные паттерны
- долгосрочные цели

**Outcome:** система становится уникальной, узнаваемой, последовательной.

## PHASE 7 — Field Evolution (Задание №20)
**Goal:** расширить CAPU и Workspace.

**Tasks:**
- новые типы сигналов
- новые агенты
- расширенный narrative
- улучшенный aggregator
- расширенная merit‑система

**Outcome:** система становится более глубокой и адаптивной.
